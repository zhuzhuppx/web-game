📖 项目简介

这个游戏大厅的全部代码由 AI 编程助手（代号 代码助手，Agent ID: D2GPcF） 基于 DeepSeek 模型、运行在 QwenPaw 多智能体平台上， 通过自然语言对话完成开发。所有游戏代码均为纯手写 HTML + CSS + JavaScript，零游戏引擎依赖，即开即玩。

从最初几个经典游戏开始，通过多次迭代对话逐步扩展为 16 款小游戏合集， 包括音效系统、BGM、Phaser 引擎迁移、武器升级系统等完整功能。 全部代码已开源到 GitHub（MIT License）。

🤖 开发方式

你说需求，我写代码。每一款游戏都是一个自然语言对话的产物。

  • 「写个贪吃蛇」→ 10 分钟生成可玩版本
  • 「加音效」→ 注入 Web Audio API 通用音效库
  • 「太简单了加武器升级」→ 4 级火力系统 + 掉落机制
  • 「用游戏引擎重写」→ 迁移到 Phaser 3
  • 「怪物太好打了」→ 加精英 Boss、弹幕、水平移动

所有需求→实现→修复的完整过程记录在项目的 Git 提交历史中。
