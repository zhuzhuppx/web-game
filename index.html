📖 项目简介

游戏大厅是一个完全由 纯手写 HTML + CSS + JavaScript 构建的网页游戏合集， 不依赖任何第三方游戏引擎或框架。每个游戏均为独立的单 HTML 文件，即开即玩。

项目最初作为开发练习产生，逐步扩展为 16 款经典小游戏合集。 所有代码完全开源，放于 GitHub 供学习交流。

🎮 游戏列表

  • 🐍 贪吃蛇 · 🧱 俄罗斯方块 · 💣 扫雷 · 🔢 2048 · 🐦 Flappy Bird
  • 👻 吃豆人 · 🔢 数独 · ⭕ 井字棋 · 🟡 四子棋 · 🃏 记忆翻牌
  • 🔨 打地鼠 · 🧱 打砖块 · 🦘 涂鸦跳跃 · 🚀 太空射击 · ♟ 中国象棋 · 🃏 斗地主

⚙️ 技术栈

  • HTML5 Canvas — 游戏渲染（贪吃蛇、俄罗斯方块、太空射击等）
  • Phaser 3 — 斗地主、中国象棋（游戏引擎版）
  • Web Audio API — 音效系统，纯代码生成，零外部音频文件
  • 响应式 + 触屏 — 同时支持桌面键盘和手机触控操作
  • CSS 动画 — 消行效果、粒子特效、UI 交互动画
